# nostr-Lists-viewer

nostrのkind30001に保存されている諸々を取得、更新したい

## 主な機能

- [x] 特定のリレーからtagごとのnoteリストを取得表示

- [x] noteIDから表示中のtagにnoteを追加

- [x] 特定のnoteを表示中のtagから削除

- [ ] 新規tagを追加

- [ ] 任意のtagを削除
